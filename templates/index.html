Project Overview

Water scarcity is becoming a major challenge due to increasing population, climate change and groundwater depletion.

This system uses a Random Forest Machine Learning model to predict whether a region is likely to face a water crisis.

Problem Statement

Many regions face water shortages because of low rainfall, excessive water consumption and declining groundwater reserves. Early prediction helps authorities take preventive actions.

Input Parameters

Parameter Description Suggested Range
Rainfall Annual rainfall received in the region 0 - 5000 mm
Population Total population depending on water resources 1,000,000,000 - 1,500,000,000
Groundwater Underground water availability indicator 50 - 21500

How Prediction Works

Step 1: User enters rainfall, population and groundwater values.

Step 2: The trained Random Forest model receives the input.

Step 3: Multiple decision trees analyze the data.

Step 4: Each decision tree votes for High Risk or Low Risk.

Step 5: Majority voting determines the final prediction.

Model Information

Algorithm Used: Random Forest Classifier

Accuracy: 100%

Training Features: Groundwater, Rainfall and Population Data

Model Storage: Pickle (.pkl)

Deployment: Flask + GitHub + Render
