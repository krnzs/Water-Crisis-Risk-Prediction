💧 AI Based Water Crisis Risk Prediction System

This application uses Machine Learning to predict the possibility of a Water Crisis in a selected region. The prediction is generated by analysing three important environmental factors: Rainfall, Population, and Groundwater Level.

The system has been trained using historical environmental data and classifies the selected region into one of two categories: High Water Crisis Risk or Low Water Crisis Risk. The prediction can assist government authorities, environmental agencies, urban planners and researchers in making informed decisions regarding water resource management.

🎯 Project Objective

The objective of this project is to develop an Artificial Intelligence based decision support system that predicts water crisis risk before it becomes severe. Early prediction enables better planning for groundwater conservation, water distribution and sustainable resource management while supporting United Nations Sustainable Development Goal (SDG 6 - Clean Water and Sanitation).

📋 How to Use

  1. Enter the Annual Rainfall of the selected region.
  2. Enter the Population of the selected region.
  3. Enter the Groundwater Level.
  4. Click Predict Water Crisis Risk.
  5. The trained Machine Learning model will analyse the inputs and display the predicted risk level.

✅ Input Validation

To improve data quality and prevent invalid predictions, the application performs both frontend and backend validation.

  • All input fields are mandatory.
  • Only numeric values are accepted.
  • Negative values are not allowed.
  • Rainfall accepts decimal values between 0 and 10000 mm.
  • Population accepts only whole numbers between 1 and 100000000.
  • Groundwater Level accepts decimal values between 0 and 500 metres.
  • If invalid data is entered, an error message is displayed instead of making a prediction.

🤖 Machine Learning Models Used

Three Machine Learning algorithms were trained and evaluated to determine the most suitable model for predicting water crisis risk.

Algorithm Purpose Accuracy
Logistic Regression Binary Classification 98.73%

Random Forest Ensemble Classification 98.73%

🏆 Selected Machine Learning Model

Although all three Machine Learning algorithms achieved an accuracy of approximately 98.73%, the Random Forest Classifier was selected as the final prediction model because it provides better generalization, reduces overfitting, and produces more reliable predictions on unseen data by combining the outputs of multiple Decision Trees.
